### What I learned

In this challenge I discovered a new way to perfectlly center a div

```
.nft-main-card{
    height: 520px;
    width: 300px;
    background: var(--very-dark-blue-card-bg);
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%,-50%);
    border-radius: 20px;
    box-shadow: 0px 0px 10px 0px rgba(128, 128, 128, 0.5);
}
```
